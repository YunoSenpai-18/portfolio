# Vincent Raphael M. Lacaste — Personal Portfolio

This repository contains my responsive personal portfolio website, built using **HTML5**, **Bootstrap 5**, and **custom CSS**. It serves as my digital resume, showcasing my biography, skills, and selected academic projects.

## Live Demo

GitHub Pages URL:  
`https://yunosenpai-18.github.io/portfolio/`

Repository URL:  
`https://github.com/YunoSenpai-18/portfolio.git`

## Tech Stack

- HTML5 (semantic structure)
- Bootstrap 5
- Custom CSS (style.css)
- Google Fonts (Inter)
- Formspree for contact form

## Features

### Hero Section

- Name, tagline, and call-to-action buttons
- Gradient hero background
- Profile card on desktop view

### About Me

- Profile photo
- Short biography and education background
- Button to connect or collaborate

### Skills

- Visual progress bars for UI/UX, Laravel, Responsive Web Design

### Projects

Three fully documented project cards, each containing:

- Title
- Screenshot/Image
- Short description
- Link to source code or live demo

Projects included:

1. **CampusCheck** – Face Recognition Attendance System
2. **Sample IRIS** – Recruitment Information System
3. **Travel Guide App** – Mobile tourism application

### Contact

- Functional contact form via Formspree
- Email, GitHub, and LinkedIn links

### Responsive Design

- Fully mobile-friendly
- Works on mobile, tablet, and desktop
- Collapsible mobile navigation menu
